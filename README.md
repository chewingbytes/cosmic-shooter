# 🚀 Cosmic Shooter

A fun and addictive space shooter game built with vanilla JavaScript, HTML5 Canvas, and CSS3.

## 🎮 Game Features

- **Classic Space Shooter Gameplay**: Defend Earth from waves of alien invaders
- **Progressive Difficulty**: Enemies get faster with each level
- **Smooth Controls**: Responsive keyboard controls with arrow keys or WASD
- **Visual Effects**: Particle explosions, glowing effects, and animated starfield
- **Score Tracking**: Keep track of your high score as you progress through levels

## 🕹️ How to Play

- **Move Left**: ← Arrow Key or A
- **Move Right**: → Arrow Key or D  
- **Shoot**: SPACE bar

## 🎯 Game Mechanics

- Destroy all enemies to advance to the next level
- Each enemy type has different point values
- Enemies speed up with each level
- Don't let enemies reach the bottom!
- You have 3 lives to achieve the highest score possible

1. Clone this repository
2. Open `index.html` in your web browser
3. Start playing!
